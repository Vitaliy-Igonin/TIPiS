# TIPiS

# Калькулятор надежности 5-элементного сечения

## Описание проекта

Веб-приложение для оценки вероятностей реализации пятиэлементного сечения для стратегии параллельного восстановления. Приложение включает:

1. Графический интерфейс для ввода параметров системы
2. Визуализацию графа состояний (32 состояния)
3. Решение системы уравнений Колмогорова-Чепмена
4. Построение графиков вероятностей состояний
5. Расчет стационарных вероятностей

## Теоретическая основа

Система состоит из 5 элементов, что дает 2⁵ = 32 возможных состояния. Для каждого состояния рассчитывается вероятность нахождения системы в нем в зависимости от времени.
